# 1: Technical Setup

## Purpose

This section documents the technical configuration required to support the reporting process within Generate.

The purpose of this section is to provide technical teams with a centralized location for configuration information, data mappings, dependencies, and migration requirements.

## Objectives

This documentation should allow technical staff to:

* Understand how source data enters Generate
* Understand migration dependencies
* Locate mapping documentation
* Review configuration settings
* Verify reporting requirements
* Support troubleshooting activities

## Technical Components

The reporting process includes:

| Component               | Description                             |
| ----------------------- | --------------------------------------- |
| Source Systems          | Systems providing reporting data        |
| Source Extracts         | Data extracted from source systems      |
| Generate Staging Tables | Initial landing area for reporting data |
| CEDS Warehouse          | Fact and dimension tables               |
| Report Tables           | Reporting output tables                 |
| EDFacts Files           | Final submission files                  |
