# Reporting Cycle Issues History

## Purpose

This page provides historical reporting information regarding issues across all reporting cycles. The <mark style="color:$warning;">Reporting Calendar</mark> shows the history and statuses of all reports.&#x20;

This page is used to track Issues that have been logged in the <mark style="color:$warning;">Report Tracking Matrix workbook</mark> and need more details.

Once an issue is determined to need more detailed information, Copy this template as save it in the Open Issues List. Once the issue is resolved it can be moved to the Resolved Issues List. Keeping this history of issues. resolutions, and decisions will help with future iterations of the reporting cycle.
