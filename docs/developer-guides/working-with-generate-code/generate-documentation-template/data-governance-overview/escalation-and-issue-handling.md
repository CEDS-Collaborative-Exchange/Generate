## Issue Categories

| Category     | Description               |
| ------------ | ------------------------- |
| Data Quality | Source data issues        |
| Migration    | ETL or migration failures |
| Validation   | Validation discrepancies  |
| Reporting    | Report generation issues  |
| Submission   | EDFacts submission issues |

## Escalation Matrix

| Issue Type        | Escalate To      | Response Target |
| ----------------- | ---------------- | --------------- |
| Data Quality      | Data Owner       | 1 Business Day  |
| Migration Failure | Technical Owner  | Same Day        |
| Validation Issue  | Validation Owner | 1 Business Day  |
| Submission Issue  | Business Owner   | Immediate       |

## Resolution Documentation

All issues should be documented in the Issue Tracking Log.

Include:

* Issue description
* Root cause
* Resolution
* Date resolved
* Lessons learned
