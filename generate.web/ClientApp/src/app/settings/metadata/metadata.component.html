Metadata
From this screen you can update the metadata that is used by Generate for all the EDFacts file specifications. This process checks for any changes or updates to the metadata and downloads those changes into Generate. The status of the most recent check is displayed below.
{{metadataStatusMessage}}
Clicking this button will initiate a metadata update process that will disable the system for the next hour. Please ensure that all data migrations are complete and you are prepared for this downtime before proceeding.
